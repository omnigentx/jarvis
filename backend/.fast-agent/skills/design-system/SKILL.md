# JARVIS DESIGN SYSTEM

## Colors
```
Primary:     #6C63FF (Indigo/Purple)
Secondary:   #2DD4BF (Teal)
Background:  #1A1A2E (Dark Navy)
Surface:     #16213E (Dark Blue)
Text:        #EAEAEA (Light Gray)
Error:       #FF6B6B (Coral Red)
Success:     #4CAF50 (Green)
Warning:     #FFB74D (Amber)
```

## Typography
- Headers: Inter Bold
- Body: Inter Regular
- Code: JetBrains Mono
- Sizes: 12 / 14 / 16 / 20 / 24 / 32px

## Spacing
- Base unit: 4px
- xs: 4px | sm: 8px | md: 16px | lg: 24px | xl: 32px | xxl: 48px

## Component Patterns
- Cards: rounded corners (12px), subtle shadow, surface background
- Buttons: min-height 40px, 8px horizontal padding
- Input fields: 1px border, 8px padding, focus ring primary color
- Agent avatar: circular, 40px, colored based on agent type

## Dark Mode (Default)
- Jarvis uses dark mode by default
- Text trên dark surface: #EAEAEA
- Borders: rgba(255,255,255,0.1)
- Hover states: lighten background 5%

## Responsive Breakpoints
- Mobile: < 768px
- Tablet: 768-1024px
- Desktop: > 1024px
